# 💪 체육관 회원관리 시스템

초보자도 바로 실행할 수 있는 체육관 회원관리 웹 애플리케이션입니다.

**기술 스택**: Python + Flask + SQLite

> **오프라인 사용 가능**: 인터넷 없이도 이 PC에서 실행할 수 있습니다. 데이터는 로컬 SQLite 파일(`gym_management.db`)에 저장됩니다. 인터넷은 Python 및 라이브러리 **최초 설치** 시에만 필요합니다.

## 📋 주요 기능

### 1. 대시보드
- 총 회원 수 표시
- 오늘 출석 수
- 3일/5일/7일 이상 미출석 회원 수
- 7일 이내 만료 회원 수

### 2. 회원 관리
- 회원 정보 등록 (이름, 전화, 생년월일, 성별)
- 회원권 종류 선택 (1/3/6/12개월)
- 만료일 자동 계산
- 회원 정보 수정 및 삭제
- 회원별 메모 기능

### 3. 출석 관리
- 회원 출석 체크
- 최근 출석일 기록
- 총 출석 횟수 통계
- 회원별 출석 기록 조회

### 4. 미출석 및 만료 관리
- 3일 이상 미출석 회원 자동 분류
- 5일 이상 미출석 회원 자동 분류
- 7일 이상 미출석 회원 자동 분류
- 7일 이내 만료 회원 목록
- 각 회원별 상세 정보 조회

## 🚀 설치 및 실행 방법

### 1단계: Python 설치 (아직 하지 않은 경우)
Python 3.9 이상이 필요합니다.
- [python.org](https://www.python.org/) 에서 다운로드하여 설치하세요.
- 설치 시 "Add Python to PATH" 옵션을 반드시 체크하세요.

### 2단계: 프로젝트 폴더에서 필수 라이브러리 설치

**Windows 명령 프롬프트 (cmd) 또는 PowerShell에서:**

```bash
# 프로젝트 폴더로 이동
cd "d:\Desktop\코파일럿 사용 예제\gym-management"

# 필수 라이브러리 설치
pip install -r requirements.txt
```

**설치 확인:**
```bash
pip list
```

Flask, SQLAlchemy 등이 설치된 것을 확인할 수 있습니다.

### 3단계: 애플리케이션 실행

**가장 쉬운 방법 (Windows):** 프로젝트 폴더의 `run.bat` 파일을 더블클릭하세요. 브라우저가 자동으로 열립니다.

또는 명령줄에서:

```bash
python app.py
```

### 4단계: 브라우저에서 접속

웹 브라우저를 열고 다음 주소로 접속하세요:

```
http://localhost:1926
```

✅ 성공하면 체육관 회원관리 대시보드가 표시됩니다!

## 💾 데이터베이스 구조

### members 테이블 (회원 정보)
```
- id: 회원 ID (자동 증가)
- name: 이름
- phone: 전화번호
- birth_date: 생년월일
- gender: 성별
- registration_date: 등록일
- membership_type: 회원권 기간 (1/3/6/12개월)
- expiry_date: 만료일
- memo: 메모
- created_at: 생성 시간
- updated_at: 수정 시간
```

### attendance 테이블 (출석 기록)
```
- id: 출석 ID (자동 증가)
- member_id: 회원 ID
- attendance_date: 출석 날짜
- check_in_time: 입실 시간
- check_out_time: 퇴실 시간
- created_at: 생성 시간
```

## 🎯 사용 방법

### 1. 신규 회원 등록
1. "회원 관리" 메뉴 → "신규 회원 등록" 버튼 클릭
2. 회원 정보 입력 (이름은 필수)
3. 회원권 기간 선택 (만료일이 자동 계산됨)
4. "등록" 버튼 클릭

### 2. 회원 정보 조회/수정
1. "회원 관리" 메뉴에서 회원 목록 확인
2. "상세" 버튼으로 상세 정보 조회
3. "수정" 버튼으로 정보 수정 가능

### 3. 출석 체크
1. "출석 관리" 메뉴 클릭
2. 회원명으로 검색 (선택사항)
3. "✅ 출석 체크" 버튼 클릭
4. 출석 기록이 자동으로 저장됨

### 4. 보고서 확인
1. "보고서" 메뉴 클릭
2. 탭을 선택하여 다음 정보 확인:
   - 7일 이내 만료 회원
   - 3일 이상 미출석 회원
   - 5일 이상 미출석 회원
   - 7일 이상 미출석 회원

## 🔧 문제 해결

### Q: "pip install 명령어가 실행되지 않습니다"
**A:** Python이 PATH에 설정되지 않은 경우입니다.
- Python을 다시 설치하면서 "Add Python to PATH" 옵션을 체크하세요.
- 또는 전체 경로로 실행: `C:\Python39\Scripts\pip install -r requirements.txt`

### Q: "포트 1926이 이미 사용 중입니다"
**A:** 다른 프로그램이 포트를 사용 중이거나, 이미 실행 중인 서버가 있습니다.
- 기존 터미널 창에서 `Ctrl + C`로 서버를 종료한 후 재시작하세요.
- 또는 `app.py` 하단의 `port=1926` 값을 다른 번호로 변경하세요.

### Q: 데이터베이스 초기화하고 싶습니다
**A:** `gym_management.db` 파일을 삭제하면 다시 생성됩니다.
- 프로그램을 종료한 후 `gym_management.db` 파일 삭제
- 프로그램 재시작 시 새로운 데이터베이스 생성
